## 🔖 專案導覽

### ✨ 現有功能

- 關鍵字觸發回覆
- 番號查詢
- 音樂播放(僅限無版權音樂)

### 🎯 相容環境

- 現代瀏覽器（Chrome >= 64, Edge >= 79, Firefox >= 78, Safari >= 12）

### 💻 本地調試

下載到本地：
```bash
$ git https://github.com/BlackishGreen33/Discord-Bot.git
$ cd Discord-Bot
```

配置環境變量：
```env
TOKEN =
APPLICATION_ID =
```

安裝依賴包：
```bash
$ yarn
```

開始調試：
```bash
$ yarn dev
```
